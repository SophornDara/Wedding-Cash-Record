# Khmer Font Installation Guide

# របៀបដំឡើងពុម្ពអក្សរខ្មែរ

## For Windows (សម្រាប់ Windows)

### Method 1: Download and Install Khmer OS Fonts

1. **Download Khmer Fonts:**
   - Visit: https://www.khmeros.info/downloads
   - Or: https://sourceforge.net/projects/khmer/files/
   - Download "KhmerOS" font package

2. **Install the Fonts:**
   - Extract the downloaded ZIP file
   - You will see files like:
     - KhmerOS.ttf
     - KhmerOS-Siemreap.ttf
     - KhmerOS-Battambang.ttf
     - KhmerOS-Bokor.ttf
   - Select all .ttf files
   - Right-click and choose "Install" or "Install for all users"

3. **Restart the Application:**
   - Close Wedding Manager completely
   - Start it again
   - Khmer text should now display correctly

### Method 2: Install via Windows Settings (Windows 10/11)

1. Press `Windows + I` to open Settings
2. Go to "Time & Language" → "Language & region"
3. Click "Add a language"
4. Search for "Khmer" or "ខ្មែរ"
5. Select "Khmer (Cambodia)" and install
6. Windows will automatically download Khmer fonts
7. Restart the application

## Font Installation Verification

After installing fonts, verify they are available:

1. Open any application like Microsoft Word or Notepad
2. Change font to "Khmer OS Siemreap" or "Khmer OS"
3. Type some Khmer text to test

## Recommended Fonts for Best Display

The application will automatically use the first available font from this list:

1. **Khmer OS Siemreap** - Best for general use
2. **Khmer OS Battambang** - Clean and readable
3. **Khmer OS** - Standard system font
4. **Hanuman** - Good for formal documents
5. **Koulen** - Bold display font

## Common Issues and Solutions

### Issue: Text still shows as boxes (□□□) after installation

**Solution:**

- Restart your computer (not just the application)
- Reinstall fonts using "Install for all users" option
- Check if fonts are in C:\Windows\Fonts folder

### Issue: Some Khmer vowels or subscripts are misaligned

**Solution:**

- This is normal for some fonts
- Try different Khmer fonts from the list above
- Update to latest Windows version which has better Unicode support

### Issue: Font looks different in different applications

**Solution:**

- This is normal - each application may render fonts slightly differently
- Use the same font family across applications for consistency

## Testing Khmer Font in Application

Once fonts are installed:

1. Open Wedding Manager
2. Look at the sidebar - you should see clear Khmer text
3. Test by entering: នាមខ្ញុំជា (My name is...)
4. If it displays correctly, fonts are working!

## Support

If you continue having font display issues after following this guide:

1. Check Windows version (older versions may have limited Unicode support)
2. Ensure Windows is fully updated
3. Try installing multiple Khmer fonts from the recommended list
4. Contact your system administrator for font installation permissions
